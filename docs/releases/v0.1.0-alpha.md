# Codex Pet League v0.1.0-alpha

This is the first public alpha release of Codex Pet League.

Codex Pet League is a Codex App and Codex CLI-first platform where users bring pets made with OpenAI's official `hatch-pet` skill into a server-authoritative pet growth and turn battle league.

## Official Shared Alpha

- Web and API: https://league.codexpetz.com
- Public assets: https://assets.codexpetz.com
- Status: https://league.codexpetz.com/status
- Support: https://league.codexpetz.com/support
- Private support email: support@codexpetz.com
- User setup guide: https://github.com/BMinChul/codex-pet-league/blob/master/docs/USER_SETUP.md

## Quick Start

```bash
git clone https://github.com/BMinChul/codex-pet-league.git
cd codex-pet-league
npm install
export CODEX_PET_LEAGUE_URL=https://league.codexpetz.com
npm run cli -- doctor
```

Sign in:

```bash
npm run cli -- auth challenge --method email_magic_link --identifier you@example.com
npm run cli -- auth verify --challenge challenge_id_from_previous_step --code EMAILCODE
export CODEX_PET_SESSION_TOKEN=league_session_token_from_verify
```

Import a real OpenAI `hatch-pet` package:

```bash
npm run cli -- pet discover-hatch
npm run cli -- pet inspect-hatch --path <hatch-pet-folder>
npm run cli -- setup --path <hatch-pet-folder> --yes --primary Forge --secondary Trace
```

Important: the first active official League pet selection is permanent for that League account. Do not lock a demo pet if you want to use a real pet later.

## Highlights

- Official shared alpha server is live on Render with Postgres, Redis, Cloudflare R2, Resend email-code login, and OpenAI moderation.
- Codex CLI, Codex App MCP bridge, and local plugin scaffold are included.
- Official `hatch-pet` package discovery, inspection, validation, import, provenance hashing, and canonical asset storage are implemented.
- Server-authoritative pet growth, XP, Style XP, levels, stats, Battle Classes, ranked LP, seasons, and ledgers are implemented.
- Real-time turn-based battles support ranked matchmaking, casual battles, friend duel invites, 30-second turn timers, AFK handling, replay logs, and ranked settlement.
- Anti-cheat and moderation controls include request guards, idempotency, replay protection, hash chains, audit recomputation, duplicate hatch source review, asset moderation states, and manual ranked locks.
- Public operating pages are available for status, support, privacy, and alpha terms.
- GitHub issue templates are available for public bug reports, install/MCP issues, and gameplay issues.

## Known Alpha Limits

- Passkeys and OAuth are intentionally deferred. The alpha uses League email-code login.
- OpenAI/ChatGPT sign-in is not treated as League account authority.
- Public issue reports must not include secrets, private source code, full Codex transcripts, session tokens, auth codes, or personal data.
- Account, moderation, privacy, and security-sensitive reports should go to support@codexpetz.com.
- The alpha may change, pause, reset, or roll back while the system is being tested.
