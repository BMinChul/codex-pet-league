CLI/MCP Install

First active League pet selection is permanent.

1. Check League npm run cli -- doctor 2. Check bridge npm run cli -- bridge status 3. Find local hatch packages npm run cli -- pet discover-hatch 4. Inspect one package npm run cli -- pet inspect-hatch --path <folder> 5. Lock official pet npm run cli -- setup --path <folder> --yes
